Operating Systems: Process Scheduling

Updated 5 minutes ago

Overview
Process scheduling determines which ready process gets CPU time next. Main goals include fairness, throughput, and low response time.

Core Algorithms
1. FCFS: easy to implement but can cause convoy effect.
2. SJF: optimal average wait time in theory, hard to predict burst accurately.
3. Round Robin: preemptive and good for interactive systems.

Exam Tip
When comparing algorithms, always reference context: batch workload vs interactive workload, and expected process burst variability.

Action Items
- Solve 3 timeline exercises.
- Review starvation prevention (aging).
- Prepare one real-world scheduler example.
